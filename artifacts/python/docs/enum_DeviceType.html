Enumerator DeviceType

Description:

Represents a particular device type as a short string.
These values apply only to DeviceStatus and DeviceInfo

Constants:

DeviceTypeGPU"gpu"Represents a graphics processing unit resource
DeviceTypeSSD"target/SSD"Represents a switchable SSD resource
DeviceTypeFPGA"fpga"Represents an FPGA resource
DeviceTypeCompute"compute"Represents a CPU resource.
