Liqid™ SDK Programming Reference for Go - Version 3.3.0

Enumerator PreDeviceType

Description:

Represents a particular device type as a short string.
These are values which apply to the PreDevice device type field.

Declaration:

type PreDeviceType string

Constants:

IdentifierValueDescription
PreDeviceTypeCompute"comp"Represents a CPU resource
PreDeviceTypeFiberChannel"fibr"Represents a fiber channel link
PreDeviceTypeFPGA"fpga"Represents an FPGA resource
PreDeviceTypeGPU"gpu"Represents a GPU resource
PreDeviceTypeInfiniband"infi"Represents an infiniband link
PreDeviceTypeLink"link"Represents a generic link
PreDeviceTypeMemory"mem"Represents switchable memory
PreDeviceTypeFabricChip"plx"Represents a PLX fabric switch chip
PreDeviceTypeTarget"targ"Represents a target resource, such as an SSD
