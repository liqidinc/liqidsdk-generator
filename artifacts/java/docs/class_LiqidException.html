Class LiqidException

Package:com.liqid.sdk

Description:

This is exception is thrown by most SDK methods to signify an unrecoverable error.
In some cases this represents a temporal condition; i.e., a retry might succeed.
In other cases this might reflect a problem with the system configuration.
Finally, it could indicate a problem in the information presented to the SDK API.
Since this class extends java.util.Exception, all accessors which apply to that class also apply to this one.
